Pack jeux Java

Développement d'un pack de mini-jeux en Java réunissant le Pendu, le Juste Prix, le Mastermind et la Bataille Navale. Chaque jeu exploite une logique dédiée : gestion de lettres avec ArrayList, génération aléatoire, analyse de séquences et manipulation de grilles. Un projet console complet, idéal pour renforcer l'algorithmie, les structures de données et la modularité du code.

Technologies utilisées

  • Java console

Site web d'un aeroport PHP POO

Conception d'un site web complet en PHP orienté objet permettant de gérer différents éléments d'un système aérien : avions, utilisateurs, rôles, vols, départs et arrivées. Le projet repose sur une architecture POO structurée en classes (Avion, Vol, Utilisateur, Role…), avec une interface dynamique en JavaScript et un design en CSS. Ce site illustre la mise en place d'un backend organisé, la gestion des permissions et la manipulation de données en MySQL. Le site n'est pas herberger sur le web, juste en local.

Technologies utilisées :

  • PHP (POO)
  • MySQL
  • JavaScript
  • HTML / CSS

Site perso E.T.H.A.N PHP

Développement d'un site web auto-hébergé sur Raspberry Pi permettant de centraliser des ressources familiales : définitions, documents, projets, recettes, calendrier, et outils personnels. Le système fonctionne en réseau local avec gestion future des rôles, authentification, modules indépendants et base de données MariaDB. Un projet complet mêlant infrastructure, backend PHP, et organisation modulaire. Le site web a pour projet d'être herberger en ligne.

Technologies utilisées

  • HTML / CSS / JavaScript
  • PHP
  • MariaDB
  • Raspberry Pi (Nginx)

Application ToDoList JavaFX

Premier projet réalisé en JavaFX, cette application ToDoList propose une interface graphique simple et ergonomique pour gérer des tâches au quotidien. Le projet intègre une base de données, permettant de sauvegarder, charger et mettre à jour les tâches de manière persistante. L'application met en avant la gestion d'événements JavaFX, la manipulation de données et la structuration propre d'un projet Java avec interface graphique.

Technologies utilisées :

  • JavaFX
  • Base de données (CRUD)
